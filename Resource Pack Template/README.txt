========================
	Notes
========================

There is no limit for the number of loading screen messages, but if the messages are too long they might go off the screen.


See inside the folder "assets\loading_screen_messages\texts":
 - The en_US.txt shows an example of a messages files.
 - You can also create messages in different languages as well. If so, you need to name the messages file accordingly like language files (For examples: see ru_RU.txt for Russian and zh_CN.txt for Chinese (Simplified)).
